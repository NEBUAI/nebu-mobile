# Changelog

Todos los cambios notables del proyecto se documentan en este archivo.

El formato se basa en [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/),
y el proyecto sigue [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2025-01-01

### Added
- Autenticacion de usuarios (login/registro)
- Conexion BLE con dispositivos Nebu
- Streaming de audio via LiveKit
- Pantalla principal con escaneo de dispositivos
- Configuracion de perfil de usuario
- Workflows de build para Android e iOS
- CI automatico en Pull Requests
